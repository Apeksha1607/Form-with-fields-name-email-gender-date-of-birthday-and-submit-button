# 📝 Personal Profile Form

A simple and modern **Personal Profile Form** built using **HTML and CSS**.
This project demonstrates how to create a clean, responsive form layout with styled input fields and a submit button.

---

## 📌 Features

* Clean and modern **UI design**
* Responsive **form container**
* Input fields for:

  * Full Name
  * Email Address
  * Gender selection
  * Date of Birth
* Styled **radio buttons**
* Gradient **submit button with hover effect**
* Simple **user-friendly layout**

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**

---

## 📷 Preview

The page includes:

* A centered **form card**
* User input fields
* Gender selection using radio buttons
* Date picker input
* Animated submit button
